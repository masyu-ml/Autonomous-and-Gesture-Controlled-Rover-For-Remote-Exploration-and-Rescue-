# Autonomous-and-Gesture-Controlled-Rover-For-Remote-Exploration-and-Rescue-
Multi-modal control using ESP32 and OpenCV, via hand gestures, or autonomously 
(does not use any sensors for navigation) it instead uses line-following and vision-based object detection to perceive depth. 
Embedded vision processing and real-time decision making.
